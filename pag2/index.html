Lista de requisitos del proyecto

  • Tres sistemas autonomos con almenos tres router internos enrutados por OSPF
  • Cada router interno debe tener a lo menos 2 segmentos LAN, con interfaz gigabitEthernet con conexion UTP
  • Cada router deve tener interface UTP y Fibra Optica
  • Use la targeta HWC-1GE-STP con el modulo GLC-LH-SMD para la coneccion por fibra
  • La coneccion entre routers debe ser atravez de fibra optica
  • Las demas conecciones atraves de UTP
  • conectar cada sistema autonomo con el protocolo BGP

Cada sistema autonomo debe incluir:

  • servicios: WEB, DNS y DHCP
  • los servivios DHCP pueden estar desde servidor o desde router(1 de cada uno minimo por sistema autonomo)
  • todos los servidores DNS deven contener la misma informacion
  • dvben existir a lo menos una pagina WEB por sistema autonomo deven ser todas distintas y solo una puede ser el ejemplo de cisco

para este Proyecto se utiliza el rango de ip 133.5.0.0 al 133.75.0.0
